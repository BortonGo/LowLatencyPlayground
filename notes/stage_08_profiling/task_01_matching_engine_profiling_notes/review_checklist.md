# Review Checklist. Matching Engine Profiling Notes

## Setup

- Указан build type.
- Указан benchmark.
- Указан инструмент profiling.
- Workload достаточно длинный для profiler.

## Notes

- Есть top hotspots.
- Есть observations.
- Есть next optimization target.
- Есть связь с benchmark цифрами.

## Engineering Thinking

- Нет оптимизации вслепую.
- Есть понимание limitations profiler output.
- Есть понимание difference between benchmark overhead and engine work.
- Следующий шаг вытекает из данных.
