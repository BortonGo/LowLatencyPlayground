# Теория. march=native

`-march=native` разрешает компилятору использовать инструкции конкретного CPU.

Плюсы:

- больше возможностей для vectorization;
- target-specific instructions;
- иногда быстрее tight loops.

Минусы:

- binary может не запускаться или работать неправильно на другом CPU;
- результаты хуже переносимы;
- на некоторых workloads выигрыша почти нет.

Это хороший flag для локального experiment, но плохой default без осознанного решения.
