# Теория. LTO

LTO означает Link Time Optimization.

Обычно компилятор оптимизирует каждый translation unit отдельно. LTO позволяет оптимизировать уже на этапе линковки, видя больше кода сразу.

Это может помочь:

- лучше inline;
- удалить unused code;
- упростить calls между `.cpp`.

Цена:

- дольше build;
- сложнее toolchain;
- иногда больше memory usage при сборке.

LTO стоит проверять отдельно, а не считать бесплатным ускорением.
