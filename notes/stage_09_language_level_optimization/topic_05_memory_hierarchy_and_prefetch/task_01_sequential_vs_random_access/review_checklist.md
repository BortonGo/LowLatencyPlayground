# Review Checklist

- Есть target `benchmark_memory_access_patterns`.
- Генерация данных не входит в measured interval.
- `sequential` и `random indexed` дают одинаковый checksum.
- Есть размеры от cache-friendly до явно больших.
- Есть таблица median results.
- Notes объясняют cache locality, а не только показывают цифры.
